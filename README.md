# LootMart Clone

An exact visual clone of [lootmart.com.pk](https://www.lootmart.com.pk/) built with vanilla HTML, CSS, and JavaScript.

## Pages

| Page | File | Description |
|------|------|-------------|
| Landing | `index.html` | Welcome hero, feature cards, promo banner, area selector, all stores |
| Store | `store.html` | Store info bar, featured products carousel, search, product grid, cart |

## Tech Stack

- **HTML5** – semantic markup
- **Tailwind CSS** (via CDN) – utility-first styling
- **Lucide Icons** (via CDN) – SVG icon library
- **Geist Font** – matching the original site's typography
- **Vanilla JavaScript** – all interactivity (cart, modals, search, area selection)

## Getting Started

Simply open `index.html` in a browser — no build step required.

```
# Or use a local server
npx serve .
```

## Screenshots

The clone replicates the full landing page and store page UIs including:

- Area selection modal (auto-shown on landing)
- Feature cards (Same-Day, Free Delivery, Multi-Store, Easy Order)
- Green promotional banner with Google sign-up CTA
- Store info blocks (hours, delivery fee, free delivery threshold)
- Featured products carousel
- Search with popular tags
- Product grid with add-to-cart functionality
- Cart sidebar with quantity controls
- WhatsApp floating button
